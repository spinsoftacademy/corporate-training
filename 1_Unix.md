* Create Directory 
    * mkdir <foldername>
* Change Directory 
    * cd <foldername>

* pwd
* List Files - ls 
* Copy Folder/Files - cp
* Move Folder/Files - mv
* Remove Folder/Files 
    * rm <filename>
    * rm -rf <foldername>
